# Web-Zakat
### Описание
Приложение с web-интерфейсом, для облечения процесса 
заказа товара в аптеку с системой товароучета "Юнико".
В программу можно загрузить заявку из "Юнико", прайс-листы
из "ePrica". Далее, уникальный алгоритм поиска найдет лучшее
соответсвие для каждой позиции в заявке. Приложение
принимает и выгружает данные в формате .dbf

### Используемые технологии
- Java Oracle OpenJDK
- Spring Boot
- PostgreSQL
- Liquibase
- Albfernandez JavaDBF
- Apache Commons
- Lombok
